# 三十六計 · 中国古典兵法智慧

一个纯静态的《三十六计》在线阅读网站，古典水墨风格设计。

## 特点

- **纯静态**：无任何后端、无构建步骤，直接用浏览器打开 `index.html` 即可。
- **完整内容**：六套共 36 计，每计包含
  - 原文（解语，含《易经》之理）
  - 难字**注音 + 注释**
  - **白话译文**
  - **按语**（原典精选）
  - **解析与典型史例**
  - **AI 绘图提示词**（水墨风，可一键复制）
- **古典水墨风**：宣纸底纹 + 墨色 + 朱砂印章 + 毛笔字体，卡片网格 + 详情弹层，支持键盘 ← → 翻计、Esc 关闭。
- **响应式**：适配桌面与移动端。

## 目录结构

```
.
├── index.html        页面结构
├── css/style.css     古典水墨风样式
├── js/data.js        36 计完整数据
├── js/app.js         渲染与交互逻辑
└── images/           配图目录（按 img-01.png … img-36.png 命名）
```

## 配图说明

每计预留了配图位 `images/img-01.png` ~ `images/img-36.png`。
**图片缺失时会自动显示水墨占位图**，不影响阅读。

每计的 AI 绘图提示词已写在 `js/data.js` 的 `prompt` 字段中，也可在网站每计详情页底部一键复制。
用你喜欢的绘图工具（Midjourney / SD / 即梦等）生成图片后，按 `img-序号.png` 命名放入 `images/` 目录即可自动显示。

## 本地预览

直接双击 `index.html`，或起一个本地服务器：

```bash
python3 -m http.server 8000
# 浏览器打开 http://localhost:8000
```
